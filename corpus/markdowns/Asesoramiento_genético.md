# Asesoramiento genético

<p>El asesoramiento genético ofrece información y apoyo a las personas que tienen o pueden tener riesgos de <a href="https://medlineplus.gov/spanish/geneticdisorders.html">trastornos genéticos</a>. Un genetista se reunirá con usted para hablar de los riesgos genéticos. La consulta puede ser sobre usted o un familiar, o puede acceder a esta información cuando piense o planee tener un bebé. Puede obtener más información con <a href="https://medlineplus.gov/spanish/genetictesting.html">pruebas genéticas</a>.</p>


<p>Existen muchas razones para buscar asesoramiento genético. Puede pensar en hacerlo si:</p><ul>
<li>	Tiene historial familiar de una afección genética o defecto congénito </li>
<li>	Está embarazada o planea estarlo después de los 35 años </li>
<li>	Ya tiene un hijo con un trastorno o defecto congénito </li>
<li>	Tuvo dos o más pérdidas de embarazos o un bebé que falleció </li>
<li>	Se realizó un ultrasonido o pruebas que sugieren la posibilidad de un problema </li>
</ul>

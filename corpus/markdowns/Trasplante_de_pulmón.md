# Trasplante de pulmón

<p>El <a href="https://medlineplus.gov/spanish/organtransplantation.html">trasplante</a> de pulmón es una cirugía para reemplazar uno o ambos pulmones enfermos por pulmones sanos de un donante. Durante la operación, el cirujano hace un corte en el pecho y extirpa el pulmón enfermo. Después, el cirujano cose el pulmón nuevo a los vasos sanguíneos principales y las vías respiratorias. </p>
<p>El médico puede recomendarle un trasplante de pulmón si tiene una enfermedad que no puede controlarse de otro modo. Estas enfermedades incluyen:</p>
<ul>
<li>	<a href="https://medlineplus.gov/spanish/copd.html">EPOC</a> (enfermedad pulmonar obstructiva crónica) </li>
<li> <a href="https://medlineplus.gov/spanish/cysticfibrosis.html">Fibrosis quística</a> </li>
<li>	<a href="https://medlineplus.gov/spanish/pulmonaryfibrosis.html">Fibrosis pulmonar</a> idiopática </li>
<li> <a href="https://medlineplus.gov/spanish/interstitiallungdiseases.html">Enfermedad pulmonar intersticial</a></li>
<li> <a href="https://medlineplus.gov/spanish/pulmonaryhypertension.html">Hipertensión pulmonar</a> primaria </li>
</ul>
<p>Las complicaciones del trasplante de pulmón incluyen rechazo del pulmón trasplantado e infecciones. </p>

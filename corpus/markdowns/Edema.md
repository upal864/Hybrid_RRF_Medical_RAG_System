# Edema

<p>Edema significa hinchazón causada por la acumulación de líquido en los tejidos del cuerpo. Suele ocurrir en los pies, los tobillos y las piernas, pero puede afectar todo el cuerpo. </p>

<p>Las causas de edema incluyen:</p>
<ul> 
<li>Consumir demasiada sal </li>
<li>Quemaduras de sol  </li>
<li>	<a href="https://medlineplus.gov/spanish/heartfailure.html">Insuficiencia cardíaca</a></li>
<li>	<a href="https://medlineplus.gov/spanish/kidneydiseases.html">Enfermedades de los riñones</a>  </li>
<li>	Problemas hepáticos por <a href="https://medlineplus.gov/spanish/cirrhosis.html">cirrosis</a> </li>
<li>	Embarazo </li>
<li>	Problemas con los <a href="https://medlineplus.gov/spanish/lymphaticdiseases.html">ganglios linfáticos</a>, especialmente después de una <a href="https://medlineplus.gov/spanish/mastectomy.html">mastectomía</a> </li>
<li>Algunas medicinas  </li>
<li>	Permanecer mucho tiempo de pie o sentado cuando hace calor  </li>
</ul>
<p>Para disminuir el edema, el médico puede recomendarle mantener sus piernas elevadas cuando se siente, usar medias elásticas, limitar el consumo de sal o tomar medicamentos llamados diuréticos.</p>

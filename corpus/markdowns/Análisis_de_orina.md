# Análisis de orina

<p>Un análisis de orina es una prueba de su orina. A menudo se realiza para chequear si hay una <a href="https://medlineplus.gov/spanish/urinarytractinfections.html">infección de las vías urinarias</a>, <a href="https://medlineplus.gov/spanish/kidneydiseases.html">problemas renales</a> o <a href="https://medlineplus.gov/spanish/diabetes.html">diabetes</a>. Usted también puede someterse a uno durante un chequeo médico, si es ingresado al hospital, antes de una cirugía o si está embarazada. Este también puede monitorear algunas afecciones médicas y tratamientos.</p>
<p>Un análisis de orina consiste en chequear:</p>
 <ul>
  <li>Su color</li>
  <li>Su aspecto (si es clara o turbia)</li>
  <li>Su olor</li>
<li>El nivel de acidez (pH)</li>
<li>Si hay sustancias que normalmente no están en la orina, como la sangre, el exceso de proteínas, glucosa, cetonas, y bilirrubina</li>
<li>Si existen células, cristales y cilindros urinarios (proteínas en forma de tubo)</li>
<li>Si presenta bacterias u otros gérmenes</li>
</ul>

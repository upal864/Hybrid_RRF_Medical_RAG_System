# Problemas respiratorios

<p>Cuando usted tiene problemas para respirar, le cuesta o es incómodo obtener el oxígeno que necesita. Puede sentir que no está obteniendo el suficiente aire.  A veces puede tener problemas respiratorios leves por una congestión nasal o hacer ejercicio muy intensamente. Pero la falta de aire también puede ser señal de una enfermedad seria. </p>
<p>Muchas condiciones pueden hacer sentir que le falta el aire:</p>

 <ul>
  <li>Problemas de los pulmones como  <a href="https://medlineplus.gov/spanish/asthma.html">asma</a>, <a href="https://medlineplus.gov/spanish/emphysema.html">enfisema</a> o <a href="https://medlineplus.gov/spanish/pneumonia.html">neumonía</a></li>
  <li>Problemas de su <a href="https://medlineplus.gov/spanish/trachealdisorders.html">tráquea</a> o en los <a href="https://medlineplus.gov/spanish/bronchialdisorders.html">bronquios</a>, que son parte de su sistema respiratorio</li>
  <li>Las <a href="https://medlineplus.gov/spanish/heartdiseases.html">enfermedades cardíacas</a> pueden provocar falta de aire si su corazón no puede bombear suficiente sangre para suministrarle oxígeno al cuerpo</li>
<li><a href="https://medlineplus.gov/spanish/anxiety.html">Ansiedad</a> o <a href="https://medlineplus.gov/spanish/panicdisorder.html">ataque de pánico</a></li>
<li><a href="https://medlineplus.gov/spanish/allergy.html">Alergias</a></li>
</ul> 
<p>Si usted tiene problemas para respirar en forma frecuente, es importante encontrar la causa.</p>

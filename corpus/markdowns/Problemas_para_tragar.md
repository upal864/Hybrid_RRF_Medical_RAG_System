# Problemas para tragar

<p>Si tiene un problema de la deglución, puede tener dificultades y dolor al tragar. Algunas personas no pueden tragar en lo absoluto o tener problemas para tragar líquidos, alimentos o saliva. Esto hace que el comer sea más difícil. Con frecuencia, puede ser difícil ingerir suficientes calorías y líquidos para alimentar al cuerpo.</p>

<p>Cualquiera puede tener un problema para tragar, pero es más común en personas mayores. En general, estos trastornos suelen ocurrir debido a otros trastornos, entre ellos:</p>
<ul>
<li>	<a href="https://medlineplus.gov/spanish/neurologicdiseases.html">Trastornos del sistema nervioso</a>: Como la <a href="https://medlineplus.gov/spanish/parkinsonsdisease.html">enfermedad de Parkinson</a> y la <a href="https://medlineplus.gov/spanish/cerebralpalsy.html">parálisis cerebral</a> </li>
<li>	Problemas en el <a href="https://medlineplus.gov/spanish/esophagusdisorders.html">esófago</a>: Incluyendo la enfermedad por reflujo gastroesofágico (<a href="https://medlineplus.gov/spanish/gerd.html">ERGE</a>) </li>
<li>	<a href="https://medlineplus.gov/spanish/stroke.html">Accidente cerebrovascular</a> </li>
<li>	Traumatismos en la <a href="https://medlineplus.gov/spanish/headinjuries.html">cabeza</a> o <a href="https://medlineplus.gov/spanish/spinalcordinjuries.html">médula espinal</a> </li>
<li>	Cáncer de <a href="https://medlineplus.gov/spanish/headandneckcancer.html">cabeza, cuello</a> o <a href="https://medlineplus.gov/spanish/esophagealcancer.html">esófago</a></li>	
</ul>
<p>Las medicinas pueden ayudar a algunas personas, mientras que otras pueden necesitar cirugía. Tal vez resulte útil modificar la dieta o sostener la cabeza o el cuello en determinada posición al comer. En los casos muy serios, las personas necesitan <a href="https://medlineplus.gov/spanish/nutritionalsupport.html">sondas para alimentarse</a>.</p>

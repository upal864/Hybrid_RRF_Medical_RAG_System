# Aparatos para movilidad asistida

<h3>¿Qué son los aparatos para movilidad asistida?</h3>

<p>Los aparatos para movilidad asistida (o dispositivos de ayuda para la movilidad) le ayudan a caminar o a moverse de un lugar a otro si tiene una <a href="https://medlineplus.gov/spanish/disabilities.html">discapacidad</a> o una lesión.</p> 

<p>Los tipos de aparatos para movilidad asistida pueden incluir:</p> 
<ul>
<li>Muletas</li>
<li>Bastones</li>
<li>Andadores</li>
<li>Sillas de ruedas</li> 
<li>Escúter motorizado</li>
<li>Prótesis, un dispositivo para reemplazar una parte faltante de su cuerpo o hacer que una parte de su cuerpo funcione mejor, como una pierna</li>
</ul>

<p>Las rampas, salvaescaleras y pasamanos también son dispositivos de ayuda para la movilidad. Estos pueden ayudarle a desplazarse de forma segura por su entorno.</p>

<h3>¿Quién puede beneficiarse de los aparatos para movilidad asistida?</h3>

<p>Los aparatos para movilidad asistida pueden ayudar a las personas que tienen una discapacidad física o problemas para desplazarse a ser más independientes. Algunas de las razones por las que un aparato para movilidad asistida podría ayudar incluyen:</p>
<ul>
<li>Es posible que necesite un andador o un bastón tiene <a href="https://medlineplus.gov/spanish/falls.html">riesgo de caerse</a> o se está recuperando de una lesión en la parte inferior del cuerpo</li> 
<li>Puede utilizar muletas para mantener el peso corporal fuera del pie, el tobillo o la rodilla.</li>
<li>Es posible que necesite una silla de ruedas o un escúter si una lesión o enfermedad le impide caminar</li> 
<li>Diversos aparatos para movilidad asistida también pueden beneficiar a personas con determinadas afecciones de salud o a adultos mayores</li>
</ul>

<h3>¿Cómo elegir un aparato para movilidad asistida?</h3>

<p>Su profesional de la salud o un fisioterapeuta pueden ayudarle a elegir un aparato para movilidad asistida. Elegir estos dispositivos requiere tiempo e investigación. Es posible que necesite buena fortaleza en la parte superior del cuerpo o buen equilibrio para usar algunos aparatos para movilidad asistida de manera segura. De lo contrario, es posible que no sean la mejor opción para usted. Debe estar preparado para usar dispositivos como muletas, bastones y andadores. Si se ajustan, estos dispositivos le brindarán apoyo, pero pueden resultar incómodos e inseguros si no lo hacen.</p>

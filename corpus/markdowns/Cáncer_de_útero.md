# Cáncer de útero

<h3>¿Qué es el cáncer de útero?</h3>
<p>El cáncer de útero se forma en los tejidos del <a href="https://medlineplus.gov/spanish/uterinediseases.html">útero</a>, también llamado matriz. El útero o matriz es el lugar donde se desarrolla el feto durante el embarazo.</p> 

<p>Existen dos tipos de cáncer de útero:</p>
<ul>
<li><strong>Cáncer de endometrio:</strong> Es el tipo más común. Este tipo de cáncer comienza en las células que recubren el útero, conocidas como endometrio.</li> 
<li><strong>Sarcoma uterino:</strong> Es un cáncer poco común, más agresivo y difícil de tratar. Este tipo de cáncer se origina en el músculo u otros tejidos del útero.</li>
</ul>

<h3>¿Quiénes tienen más probabilidades de desarrollar cáncer de útero?</h3>
<p>El cáncer de útero suele aparecer después de la menopausia. Se desconoce su causa. Sin embargo, algunos factores y afecciones que alteran el equilibrio hormonal del cuerpo pueden aumentar el riesgo. Estos incluyen la obesidad o el <a href="https://medlineplus.gov/spanish/metabolicsyndrome.html">síndrome metabólico</a> (un grupo de factores de riesgo para ciertos problemas de salud).</p>

<p>Otros factores y afecciones que pueden aumentar el riesgo de padecer:</p>

<p><strong>Cáncer de endometrio</strong></p>
<ul> 
<li>Recibir <a href="https://medlineplus.gov/spanish/hormonereplacementtherapy.html">terapia de reemplazo hormonal</a> (TRH) solo con estrógenos después de la menopausia</li> 
<li>Tener <a href="https://medlineplus.gov/spanish/diabetes.html">diabetes</a>tipo 2</li> 
<li>Comenzar la menstruación a una edad temprana o la menopausia a una edad más avanzada</li>
<li>No haber estado nunca embarazada</li>
<li>Tomar tamoxifeno, un medicamento utilizado para prevenir o tratar el cáncer de mama</li> 
<li>Tener <a href="https://medlineplus.gov/spanish/polycysticovarysyndrome.html">síndrome de ovario poliquístico</a></li>
<li>Tener una madre, hermana o hija que haya tenido cáncer de endometrio</li>
<li>Tener ciertas afecciones genéticas, como el síndrome de Lynch</li>
<li>Tener hiperplasia endometrial (engrosamiento del revestimiento uterino)</li>
</ul>

<p><strong>Sarcoma uterino</strong></p>
<ul>
<li>Haber recibido tratamiento previo con <a href="https://medlineplus.gov/spanish/radiationtherapy.html">radioterapia</a> pélvica</li>
<li>Tomar tamoxifeno, un medicamento utilizado para prevenir o tratar el cáncer de mama</li> 
</ul>

<h3>¿Cuáles son los síntomas del cáncer de útero?</h3>
<p>El síntoma más común tanto del cáncer de endometrio como del sarcoma uterino es el <a href="https://medlineplus.gov/spanish/vaginalbleeding.html">sangrado vaginal</a> anormal. Ambos tipos de cáncer uterino también pueden causar <a href="https://medlineplus.gov/spanish/pelvicpain.html">dolor pélvico</a> o presión, flujo vaginal inusual o agrandamiento del útero o una masa pélvica.</p>

<p>Los síntomas menos comunes pueden incluir orinar con frecuencia, tener dificultad para orinar o dolor durante las relaciones sexuales.</p> 

<h3>¿Cómo se diagnostica el cáncer de útero?</h3>
<p>Si tiene síntomas de cáncer de útero, su profesional de la salud podría:</p>
<ul>
<li>Preguntarle sobre sus antecedentes médicos e <a href="https://medlineplus.gov/spanish/familyhistory.html">historia familiar</a></li>
<li>Realizarle un <a href="https://medlineplus.gov/spanish/womenshealthcheckup.html">examen pélvico</a></li>
<li>Solicitar pruebas de <a href="https://medlineplus.gov/spanish/diagnosticimaging.html">diagnóstico por imagen</a></li>
<li>Sugerirle una <a href="https://medlineplus.gov/spanish/biopsy.html">biopsia</a> o un procedimiento menor llamado dilatación y legrado (D y L) para revisar el endometrio</li> 
</ul>

<h3>¿Cuáles son los tratamientos para el cáncer de útero?</h3>
<p>El tratamiento puede depender de su salud, la extensión del cáncer y si se ha propagado. El tratamiento más común es una <a href="https://medlineplus.gov/spanish/hysterectomy.html">histerectomía</a>, que es una cirugía para extirpar el útero. A veces, la cirugía también extrae los ovarios y las trompas de Falopio.</p> 

<p>Es posible que reciba más de un tipo de tratamiento. Otros tratamientos pueden incluir:</p>
<ul>
<li>Terapia hormonal</li>
<li>Radioterapia</li>
<li><a href="https://medlineplus.gov/spanish/cancerchemotherapy.html">Quimioterapia</a></li> 
</ul>

<h3>¿Se puede prevenir el cáncer de útero?</h3>
<p>No existe una forma segura de prevenir el cáncer de útero. Sin embargo, puede tomar medidas para reducir el riesgo, como mantener un <a href="https://medlineplus.gov/spanish/weightcontrol.html">peso saludable</a>, realizar <a href="https://medlineplus.gov/spanish/howmuchexercisedoineed.html">actividad física con regularidad</a> y hablar con su profesional de la salud sobre los beneficios y riesgos de la terapia hormonal.</p>

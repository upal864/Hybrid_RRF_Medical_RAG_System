# Enfermedades neurológicas

<p>El cerebro, la médula espinal y los nervios conforman el sistema nervioso. En conjunto controlan todas las funciones del cuerpo. Cuando algo funciona mal en una parte del sistema nervioso, es posible que tenga dificultad para moverse, hablar, tragar, respirar o aprender. También puede haber problemas con la memoria, los sentidos o el estado de ánimo. </p>
<p>Existen más de 600 enfermedades neurológicas. Los tipos más reconocidos incluyen:</p>
<ul>
<li>Enfermedades causadas por genes defectuosos, tales como la <a href="https://medlineplus.gov/spanish/huntingtonsdisease.html">enfermedad de Huntington</a> y la <a href="https://medlineplus.gov/spanish/musculardystrophy.html">distrofia muscular</a></li>
<li>Problemas con el desarrollo del sistema nervioso, tales como la <a href="https://medlineplus.gov/spanish/spinabifida.html">espina bífida</a></li>
<li><a href="https://medlineplus.gov/spanish/degenerativenervediseases.html">Enfermedades degenerativas</a>, en las cuales las células nerviosas están dañadas o mueren, tales como las enfermedades de <a href="https://medlineplus.gov/spanish/parkinsonsdisease.html">Parkinson</a> y el <a href="https://medlineplus.gov/spanish/alzheimersdisease.html">mal de Alzheimer</a></li>
<li>Enfermedades de los vasos sanguíneos que abastecen el cerebro, tales como los <a href="https://medlineplus.gov/spanish/stroke.html">derrames cerebrales</a></li>
<li>Lesiones en la <a href="https://medlineplus.gov/spanish/spinalcordinjuries.html">médula espinal</a> y el <a href="https://medlineplus.gov/spanish/traumaticbraininjury.html">cerebro</a></li>
<li>Trastornos convulsivos, tales como la <a href="https://medlineplus.gov/spanish/epilepsy.html">epilepsia</a></li>
<li>Cáncer, tales como los <a href="https://medlineplus.gov/spanish/braintumors.html">tumores cerebrales</a></li>
<li>Infecciones como la <a href="https://medlineplus.gov/spanish/meningitis.html">meningitis</a></li>
</ul>

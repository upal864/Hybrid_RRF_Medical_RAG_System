# Enfermedades musculares

<p>Los músculos lo ayudan a moverse y forman parte del funcionamiento del cuerpo. Los distintos tipos de músculos tienen funciones diferentes. Existen muchos problemas que pueden afectar a los músculos. Las enfermedades musculares pueden causar debilidad, dolor o inclusive parálisis. </p>

<p>Algunas causas conocidas son:</p><ul>
<li>	Lesiones o exceso de uso, como en <a href="https://medlineplus.gov/spanish/sprainsandstrains.html">torceduras y distensiones</a>, <a href="https://medlineplus.gov/spanish/musclecramps.html">calambres</a> o <a href="https://medlineplus.gov/spanish/tendinitis.html">tendinitis</a> </li>
<li>	Genéticas, como la <a href="https://medlineplus.gov/spanish/musculardystrophy.html">distrofia muscular</a> </li>
<li>	<a href="https://medlineplus.gov/spanish/softtissuesarcoma.html">Cáncer</a> </li>
<li>	Inflamación, como la <a href="https://medlineplus.gov/spanish/myositis.html">miositis</a>
 </li>
<li>	Enfermedades de los <a href="https://medlineplus.gov/spanish/neuromusculardisorders.html">nervios</a> que afectan a los músculos </li>
<li>	Infecciones </li>
<li>	Algunas medicinas </li>
</ul>

<p>En algunas ocasiones, no se conoce la causa de las enfermedades musculares. </p>

# Obstrucción intestinal

<p>Una obstrucción intestinal ocurre cuando la comida o las <a href="https://medlineplus.gov/spanish/bowelmovement.html">heces</a> no pueden salir del intestino. La obstrucción puede ser completa o parcial. Hay muchas causas. Las más comunes son <a href="https://medlineplus.gov/spanish/adhesions.html">adherencias</a>, <a href="https://medlineplus.gov/spanish/hernia.html">hernias</a>, cánceres y algunas medicinas.</p>
<p>Los síntomas incluyen:</p>
<ul>
<li>Dolor abdominal severo o retorcijones </li>
<li>Vómitos </li>
<li>Inflamación </li>
<li>Gases fuertes </li>
<li>Hinchazón del vientre </li>
<li>Incapacidad para eliminar gases </li>
<li><a href="https://medlineplus.gov/spanish/constipation.html">Estreñimiento</a></li>
</ul>
<p>Una obstrucción intestinal completa es una emergencia médica. En general, se necesita cirugía.</p>

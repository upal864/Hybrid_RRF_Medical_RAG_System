# Picazón

<h3>¿Qué es la picazón?</h3>
<p>La picazón es una sensación irritante que despierta la necesidad de rascarse la piel. A veces puede sentirse como algo parecido al dolor, pero es diferente. A menudo, siente que le pica una parte del cuerpo, pero a veces puede sentir comezón en todo su cuerpo. Junto con la picazón, también puede tener <a href="https://medlineplus.gov/spanish/rashes.html">sarpullido</a> o <a href="https://medlineplus.gov/spanish/hives.html">urticaria</a>.</p>

<h3>¿Qué causa picazón?</h3>

<p>La picazón es un síntoma de muchas afecciones de salud. Algunas causas comunes son:</p>

<ul>
  <li><a href="https://medlineplus.gov/spanish/allergy.html">Reacciones alérgicas</a> a <a href="https://medlineplus.gov/spanish/foodallergy.html">alimentos</a>, <a href="https://medlineplus.gov/spanish/insectbitesandstings.html">picaduras de insectos</a>, <a href="https://medlineplus.gov/spanish/hayfever.html">polen</a> y <a href="https://medlineplus.gov/spanish/drugreactions.html">medicamentos</a></li>
  <li><a href="https://medlineplus.gov/spanish/skinconditions.html">Enfermedades de la piel</a> como <a href="https://medlineplus.gov/spanish/eczema.html">eccema</a>, <a href="https://medlineplus.gov/spanish/psoriasis.html">psoriasis</a> y piel seca</li>
  <li>Productos químicos irritantes, <a href="https://medlineplus.gov/spanish/cosmetics.html">cosméticos</a> y otras sustancias</li>
<li>Parásitos como <a href="https://medlineplus.gov/spanish/pinworms.html">oxiuros</a>, <a href="https://medlineplus.gov/spanish/scabies.html">sarna</a>, <a href="https://medlineplus.gov/spanish/headlice.html">piojos de la cabeza</a> y del <a href="https://medlineplus.gov/spanish/bodylice.html">cuerpo</a></li>
  <li>El embarazo</li>
  <li>Enfermedades del <a href="https://medlineplus.gov/spanish/liverdiseases.html">hígado</a>, <a href="https://medlineplus.gov/spanish/kidneydiseases.html">riñón</a> o <a href="https://medlineplus.gov/spanish/thyroiddiseases.html">tiroides</a></li>
<li>Ciertos tipos de <a href="https://medlineplus.gov/spanish/cancer.html">cáncer</a> o tratamientos contra el cáncer</li>
  <li>Enfermedades que pueden afectar el sistema nervioso, como la <a href="https://medlineplus.gov/spanish/diabetes.html">diabetes</a> y la <a href="https://medlineplus.gov/spanish/shingles.html">culebrilla</a> (herpes zóster)</li>
</ul>

<h3>¿Cuáles son los tratamientos para la picazón?</h3>
<p>La mayoría de las picazones no son serias. Para sentirse mejor, puede intentar:</p>

<ul>
  <li>Aplicar compresas frías</li>
  <li>Usar lociones humectantes</li>
  <li>Tomar baños tibios o de avena</li>
<li>Usar crema de hidrocortisona o antihistamínicos sin receta</li>
  <li>Evitar los arañazos, el uso de telas irritantes y la exposición a altas temperaturas y humedad</li>
</ul>


<p>Hable con su proveedor de atención médica si su picazón es severa, no desaparece después de algunas semanas o no tiene una causa aparente. Es posible que necesite otros tratamientos, como medicamentos o fototerapia. Si tiene una enfermedad subyacente que está causando la picazón, tratar esa enfermedad puede ayudar.</p>

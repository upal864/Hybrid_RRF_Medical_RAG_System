# Preparativos y recuperación ante un desastre

<p>La preparación para un desastre puede disminuir el miedo, la ansiedad y las pérdidas que puede causar el desastre. Un desastre puede ser un desastre natural, como un <a href="https://medlineplus.gov/spanish/hurricanes.html">huracán</a>, un <a href="https://medlineplus.gov/spanish/tornadoes.html">tornado</a>, una <a href="https://medlineplus.gov/spanish/floods.html">inundación</a> o un <a href="https://medlineplus.gov/spanish/earthquakes.html">terremoto</a>. También puede ser provocado por el hombre, como un <a href="https://medlineplus.gov/spanish/biodefenseandbioterrorism.html">ataque terrorista</a> o un <a href="https://medlineplus.gov/spanish/chemicalemergencies.html">derrame de sustancias químicas</a>. Debe conocer los signos de riesgo y peligro de los distintos tipos de desastres. También debe tener un plan ante un desastre. Asegúrese de evacuar su casa y saber cómo tratar problemas médicos básicos. Asegúrese también de tener el seguro que necesita, inclusive algunos tipos especiales, como el seguro contra inundaciones.</p>

<p>Independientemente del tipo de desastre que ocurra, siempre provoca un <a href="https://medlineplus.gov/spanish/copingwithdisasters.html">disturbio emocional</a>. La recuperación después de un desastre puede llevar cierto tiempo. Manténgase en contacto con sus familiares y amigos durante ese período.</p>

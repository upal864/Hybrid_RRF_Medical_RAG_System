# Leucodistrofias

<h3>¿Qué son las leucodistrofias?</h3>

<p>Las leucodistrofias son un grupo de <TOPIC ID="3118" LINKTEXT="trastornos 
 genéticos"/> raros que afectan el sistema nervioso central, el que está formado por su cerebro y médula espinal. Las leucodistrofias dañan la sustancia blanca (o materia blanca) de su sistema nervioso central. La sustancia blanca incluye:</p>

<ul>
  <li>Fibras nerviosas: Conectan las células nerviosas, también se conocen como axones</li>
  <li>Mielina: Capa de proteínas y materia grasa que recubre y protege las fibras nerviosas. También ayuda a acelerar las señales entre las células nerviosas</li>
 </ul>


<p>Cuando la sustancia blanca está dañada, puede hacer más lentas o bloquear las señales entre las células nerviosas. Esto puede causar muchos síntomas diferentes, incluyendo problemas de <a href="https://medlineplus.gov/spanish/movementdisorders.html">movimiento</a>, <a href="https://medlineplus.gov/spanish/visionimpairmentandblindness.html">vista</a>, <a href="https://medlineplus.gov/spanish/hearingdisordersanddeafness.html">audición </a>y pensamiento.</p>

<p>Hay más de 50 tipos de leucodistrofias. Algunos tipos están presentes al nacer, mientras que otros pueden no causar síntomas hasta que el niño comienza a caminar. Algunos tipos afectan principalmente a los adultos. La mayoría de los tipos empeoran con el tiempo.</p>

<h3>¿Qué causa las leucodistrofias?</h3>
<p>Las leucodistrofias son causadas por cambios genéticos. En general, estos cambios se heredan, lo que significa que se transmiten de padres a hijos.</p>

<h3>¿Cuáles son los síntomas de las leucodistrofias?</h3>
<p>Los síntomas de las leucodistrofias dependen del tipo. Pueden incluir una pérdida gradual de:</p>

<ul>
 <li>Tono muscular</li>
  <li> <a href="https://medlineplus.gov/spanish/balanceproblems.html">Equilibrio</a> y movilidad</li>
  <li>El caminar</li>
  <li>El habla</li>
<li>Habilidad para comer</li>
  <li>La visión</li>
  <li>El escuchar</li>
  <li>El comportamiento</li>
</ul>

<p>También puede haber otros síntomas, como:</p>
<ul>
  <li><a href="https://medlineplus.gov/spanish/learningdisabilities.html">Problemas del aprendizaje</a></li>
  <li><a href="https://medlineplus.gov/spanish/bladderdiseases.html">Problemas de la vejiga</a></li>
  <li><a href="https://medlineplus.gov/spanish/breathingproblems.html">Problemas respiratorios</a></li>
<li><a href="https://medlineplus.gov/spanish/developmentaldisabilities.html">Problemas del desarrollo</a></li>
  <li>Trastornos del control muscular</li>
  <li><a href="https://medlineplus.gov/spanish/seizures.html">Convulsiones</a></li>
</ul>


<h3>¿Cómo se diagnostican las leucodistrofias?</h3>
<p>Las leucodistrofias pueden ser difíciles de diagnosticar porque hay muchos tipos diferentes que pueden tener síntomas distintos. Su profesional de la salud puede hacer un diagnóstico al evaluar:</p>
<ul>
  <li>Exámenes físicos y <a href="https://medlineplus.gov/spanish/pruebas-de-laboratorio/examen-neurologico/">neurológicos</a> </li>
  <li>Su historia clínica, incluyendo preguntas sobre antecedentes familiares</li>
  <li><a href="https://medlineplus.gov/spanish/diagnosticimaging.html">Pruebas de imágenes</a>, como una <a href="https://medlineplus.gov/spanish/mriscans.html">resonancia magnética</a> o una <a href="https://medlineplus.gov/spanish/ctscans.html">tomografía computarizada</a></li>
 <li><a href="https://medlineplus.gov/spanish/genetictesting.html">Pruebas genéticas</a> para buscar cambios genéticos que podrían causar leucodistrofias</li>
  <li>Pruebas de laboratorio</li>
</ul>


<h3>¿Cuáles son los tratamientos para las leucodistrofias?</h3>
<p>No existe cura para las leucodistrofias. El tratamiento busca aliviar los síntomas y brindar apoyo. Puede incluir:</p>

<ul>
  <li>Medicamentos para controlar el tono muscular, las convulsiones y la rigidez muscular (espasticidad)</li>
  <li>Terapias físicas, ocupacionales y del habla para mejorar la movilidad, la función y los problemas cognitivos</li>
  <li>Terapia nutricional para problemas de alimentación y para tragar</li>
 <li>Programas educativos y recreativos</li>
</ul>


<p>El trasplante de <a href="https://medlineplus.gov/spanish/stemcells.html">células madre</a> o de <a href="https://medlineplus.gov/spanish/bonemarrowtransplantation.html">médula ósea</a> puede ayudar en algunos tipos de leucodistrofia.</p>

<p>Un tipo de leucodistrofia, xantomatosis cerebrotendinosa (CTX), es tratable si se diagnostica temprano. Se trata con terapia de reemplazo de ácido quenodesoxicólico.</p>

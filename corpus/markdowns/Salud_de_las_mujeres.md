# Salud de las mujeres

<p>Las mujeres tienen problemas de salud únicos. Y algunos de los problemas de salud que afectan a hombres y mujeres pueden afectar a estas últimas de manera diferente. </p>

<p>Temas únicos incluyen el <a href="https://medlineplus.gov/spanish/pregnancy.html">embarazo</a>, la <a href="https://medlineplus.gov/spanish/menopause.html">menopausia</a> y las afecciones de los órganos femeninos. Las mujeres pueden tener un embarazo saludable si reciben <a href="https://medlineplus.gov/spanish/prenatalcare.html">cuidado prenatal</a> temprano y regular. También deben hacerse las pruebas recomendadas para el cáncer de mama, <a href="https://medlineplus.gov/spanish/cervicalcancerscreening.html">cervical</a> y de <a href="https://medlineplus.gov/spanish/bonedensity.html">densidad ósea</a>. </p>

<p>Mujeres y hombres también tienen muchos problemas de salud similares. Sin embargo, estos problemas pueden afectar a las mujeres de manera diferente. Por ejemplo:</p>


<ul>
<li>	Las mujeres tienen más probabilidades de morir a causa de un <a href="https://medlineplus.gov/spanish/heartattack.html">ataque al corazón</a> que los hombres </li>
<li>	Las mujeres tienen más probabilidades de mostrar signos de <a href="https://medlineplus.gov/spanish/depression.html">depresión</a> y <a href="https://medlineplus.gov/spanish/anxiety.html">ansiedad</a> que los hombres </li>
<li>	Los efectos de las infecciones de transmisión sexual (ITS) pueden ser más graves en las mujeres. </li>
<li>La <a href="https://medlineplus.gov/spanish/osteoarthritis.html">osteoartritis</a> afecta más a las mujeres que a los hombres </li>
<li>Las mujeres tienen más probabilidades de tener <a href="https://medlineplus.gov/spanish/urinarytractinfections.html">infecciones de las vías urinarias</a> </li>
</ul>

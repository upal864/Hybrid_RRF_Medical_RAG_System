# Riesgos para el sistema reproductivo

<h3>¿Qué son los riesgos para el sistema reproductivo?</h3>

<p>Los riesgos para el sistema reproductivo son sustancias que pueden afectar la salud reproductiva de mujeres y hombres. También incluyen sustancias que afectan la capacidad delas personas de tener hijos sanos. Estas sustancias pueden ser químicas, físicas o biológicas. Algunos tipos comunes incluyen:</p>

<ul>
  <li><a href="https://medlineplus.gov/spanish/alcohol.html">Alcohol</a></li>
  <li>Químicos como <a href="https://medlineplus.gov/spanish/pesticides.html">pesticidas</a></li>
  <li><a href="https://medlineplus.gov/spanish/smoking.html">Tabaco</a></li>
<li><a href="https://medlineplus.gov/spanish/drugreactions.html">Drogas</a> legales e <a href="https://medlineplus.gov/spanish/druguseandaddiction.html">ilegales</a></li>
<li>Metales como el <a href="https://medlineplus.gov/spanish/leadpoisoning.html">plomo</a> y el <a href="https://medlineplus.gov/spanish/mercury.html">mercurio</a></li>
  <li><a href="https://medlineplus.gov/spanish/radiationexposure.html">Radiación</a></li>
  <li>Algunos <a href="https://medlineplus.gov/spanish/viralinfections.html">virus</a></li>
</ul>

<p>Se puede exponer a estos riesgos a través del contacto con la piel, la respiración o tragarlos. Esto puede ocurrir en cualquier parte, pero es más común en el lugar de trabajo o en el hogar.</p>
   
<h3>¿Cuáles son los efectos de los riesgos para el sistema reproductivo?</h3>

<p>Los posibles efectos en la salud de los riesgos para el sistema reproductivo incluyen <a href="https://medlineplus.gov/spanish/infertility.html">fertilidad</a>, <a href="https://medlineplus.gov/spanish/miscarriage.html">pérdida del embarazo</a>, <a href="https://medlineplus.gov/spanish/birthdefects.html">defectos congénitos</a> y <a href="https://medlineplus.gov/spanish/developmentaldisabilities.html">trastornos del desarrollo</a> en niños. El tipo de efectos en la salud y qué tan serios son dependen de muchos factores, como:</p>
<ul>
  <li>El tipo de sustancia </li>
  <li>La cantidad a la que estuvo expuesto</li>
  <li>Cuánto y cuán seguido estuvo expuesto</li>
<li>Cómo reaccionó a la sustancia</li>
</ul>

<h3>¿Cómo los riesgos para el sistema reproductivo afectan a los hombres?</h3>

<p>Una sustancia puede afectar los espermatozoides en los hombres, ya sea la cantidad, su forma o cómo nadan. También puede dañar su ADN. Puede que no sea capaz de fecundar el óvulo, o que cause problemas con el <a href="https://medlineplus.gov/spanish/fetalhealthanddevelopment.html">desarrollo del feto</a>.</p>

<h3>¿Cómo los riesgos para el sistema reproductivo afectan a las mujeres?</h3>
<p>Para la mujer, una sustancia riesgosa para el sistema reproductivo puede afectar su <a href="https://medlineplus.gov/spanish/menstruation.html">ciclo menstrual</a>. Puede causar un desequilibrio hormonal, lo que puede aumentar el riesgo de enfermedades como la <a href="https://medlineplus.gov/spanish/osteoporosis.html">osteoporosis</a>, <a href="https://medlineplus.gov/spanish/heartdiseaseinwomen.html">enfermedad cardiaca</a> y ciertos tipos de cáncer. También puede afectar la capacidad de una <a href="https://medlineplus.gov/spanish/femaleinfertility.html">mujer de quedar embarazada</a>. </p>

<p>Una mujer expuesta a estos riesgos durante el embarazo puede tener diferentes efectos, dependiendo de cuándo estuvo expuesta. Durante los primeros 3 meses de gestación, puede causar defectos congénitos o pérdida del embarazo. Durante los últimos 6 meses de gestación, puede retrasar el crecimiento del feto, afectar el desarrollo de su cerebro o llevar a <a href="https://medlineplus.gov/spanish/pretermlabor.html">trabajo de parto prematuro</a>. </p>

<h3>¿Cómo se pueden evitar los riesgos para el sistema reproductivo?</h3>

<p>Para intentar evitar peligros reproductivos:</p>
<ul>
  <li>Evite el alcohol y las drogas ilegales <a href="https://medlineplus.gov/spanish/pregnancyandsubstanceuse.html">durante el embarazo</a></li>
  <li>Si fuma, trate de <a href="https://medlineplus.gov/spanish/quittingsmoking.html">dejarlo</a>. Y si no fuma, no empiece a fumar</li>
  <li>Tome precauciones si está utilizando productos químicos <a href="https://medlineplus.gov/spanish/householdproducts.html">domésticos</a> o pesticidas</li>
<li>Use buena <a href="https://medlineplus.gov/spanish/germsandhygiene.html">higiene</a>, incluyendo lavado de manos</li>
  <li>Si hay riesgos en su <a href="https://medlineplus.gov/spanish/occupationalhealth.html">trabajo</a>, asegúrese de seguir prácticas y procedimientos de trabajo seguros
</li>
</ul>

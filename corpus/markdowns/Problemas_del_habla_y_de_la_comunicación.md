# Problemas del habla y de la comunicación

<p>Muchos problemas pueden afectar nuestra capacidad para hablar y comunicarnos. Varían desde decir las palabras de manera incorrecta hasta la incapacidad total para hablar o entender el habla. Entre las causas se incluyen:</p><ul>
<li>	<a href="https://medlineplus.gov/spanish/hearingdisordersanddeafness.html">Problemas auditivos y sordera</a> </li>
<li> <a href="https://medlineplus.gov/spanish/voicedisorders.html">Problemas con la voz</a>, como la disfonía o los problemas causados por el <a href="https://medlineplus.gov/spanish/cleftlipandpalate.html">labio leporino o paladar hendido</a></li>
<li>	Problemas del habla, como el <a href="https://medlineplus.gov/spanish/stuttering.html">tartamudeo</a> </li>
<li>	<a href="https://medlineplus.gov/spanish/developmentaldisabilities.html">Discapacidades del desarrollo</a> </li>
<li>	<a href="https://medlineplus.gov/spanish/learningdisabilities.html">Problemas de aprendizaje</a> </li>
<li>	<a href="https://medlineplus.gov/spanish/autismspectrumdisorder.html">Trastorno del espectro autista</a></li>
<li>	<a href="https://medlineplus.gov/spanish/traumaticbraininjury.html">Lesión cerebral</a> </li>
</ul>
<p>Algunos problemas del habla y la comunicación pueden ser genéticos. Frecuentemente, se desconocen las causas. En el primer grado de enseñanza, aproximadamente 5% de los <a href="https://medlineplus.gov/spanish/speechandlanguageproblemsinchildren.html">niños</a> tienen problemas del habla notorios. La terapia del habla y del lenguaje puede serles útil.</p>

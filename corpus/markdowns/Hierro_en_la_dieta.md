# Hierro en la dieta

<h3>¿Qué es el hierro?</h3>
<p>El hierro es un <a href="https://medlineplus.gov/spanish/minerals.html">mineral</a> que el cuerpo necesita para su crecimiento y desarrollo. Su cuerpo usa hierro para producir hemoglobina, una proteína en los glóbulos rojos. La hemoglobina transporta oxígeno desde los pulmones a todas las partes del cuerpo. El hierro también es importante para la salud de los músculos, la médula ósea y el funcionamiento de los órganos. Su cuerpo también lo necesita para producir algunas <a href="https://medlineplus.gov/spanish/hormones.html">hormonas</a>.</p>

<h3>¿Cómo se obtiene el hierro?</h3>
<p>El hierro se encuentra naturalmente en muchos alimentos y se agrega a algunos productos alimenticios enriquecidos. Los alimentos con alto contenido de hierro incluyen:</p>
<ul>
<li>Carnes magras, mariscos y aves</li>
<li>Panes y cereales para el desayuno fortificados con hierro</li>
<li>Frijoles blancos, lentejas, espinacas, frijoles rojos ("kidney") y guisantes</li>
<li>Nueces y algunas frutas secas, como pasas</li>
</ul>

<p>El hierro también se puede obtener en suplementos, ya sea solo o como parte de muchos multivitamínicos/minerales.</p>

<h3>¿Qué causa la falta de hierro?</h3>
<p>La mayoría de las personas en los Estados Unidos obtiene suficiente hierro. La cantidad que necesita cada día depende de su edad, sexo y si consume una dieta principalmente basada en plantas.</p>

<p>En ocasiones, las personas pueden tener problemas para obtener suficiente hierro. Esto puede tener muchas causas, incluyendo pérdida de sangre, una mala alimentación o un problema para absorber suficiente hierro de los alimentos. Quienes tienen más probabilidades de tener niveles bajos de hierro son aquellas personas que:</p>
<ul>
<li>Tienen <a href="https://medlineplus.gov/spanish/menstruation.html">menstruaciones</a> abundantes</li>
<li>Están <a href="https://medlineplus.gov/spanish/pregnancy.html">embarazadas</a> o <a href="https://medlineplus.gov/spanish/breastfeeding.html">amamantando</a></li>
<li>Son bebés (especialmente si fueron <a href="https://medlineplus.gov/spanish/prematurebabies.html">prematuros</a> o con <a href="https://medlineplus.gov/spanish/birthweight.html">bajo peso al nacer</a>)</li>
<li>Son <a href="https://medlineplus.gov/spanish/bloodtransfusionanddonation.html">donantes de sangre</a> frecuentes</li>
<li>Tienen <a href="https://medlineplus.gov/spanish/cancer.html">cáncer</a>, ciertas <a href="https://medlineplus.gov/spanish/digestivediseases.html">enfermedades digestivas</a> o <a href="https://medlineplus.gov/spanish/heartfailure.html">insuficiencia cardíaca&lt;</a>/li&gt;
</li><li>Requieren <a href="https://medlineplus.gov/spanish/dialysis.html">diálisis renal</a></li>
<li>Tienen problemas para absorber el hierro porque:
  <ul>
  <li>Tienen una afección digestiva como <a href="https://medlineplus.gov/spanish/celiacdisease.html">enfermedad celíaca</a>, <a href="https://medlineplus.gov/spanish/ulcerativecolitis.html">colitis ulcerosa</a>, <a href="https://medlineplus.gov/spanish/crohnsdisease.html">enfermedad de Crohn </a>o <a href="https://medlineplus.gov/spanish/helicobacterpyloriinfections.html">infección por Helicobacter pylori</a></li>
  <li>Tuvieron una <a href="https://medlineplus.gov/spanish/weightlosssurgery.html">cirugía para bajar de peso</a></li>
  </ul>
</li>
</ul>

<h3>¿Qué ocurre si no obtiene suficiente hierro?</h3>
<p>Si tiene muy poco hierro, puede desarrollar <a href="https://medlineplus.gov/spanish/anemia.html">anemia</a> por deficiencia de hierro. Puede que al principio no cause síntomas, pero con el tiempo puede provocar <a href="https://medlineplus.gov/spanish/fatigue.html">fatiga</a>, <a href="https://medlineplus.gov/spanish/breathingproblems.html">dificultad para respirar</a> y problemas con la <a href="https://medlineplus.gov/spanish/memory.html">memoria</a> y la concentración.</p>
<p>La anemia por deficiencia de hierro y los niveles bajos de hierro se suele tratar con suplementos de hierro.</p>

<h3>¿Qué pasa si obtiene demasiado hierro?</h3>
<p>Demasiado hierro puede dañar su cuerpo. Por ejemplo, si está sano y toma demasiados suplementos de hierro, puede tener síntomas como <a href="https://medlineplus.gov/spanish/constipation.html">estreñimiento</a>, <a href="https://medlineplus.gov/spanish/nauseaandvomiting.html">náusea y vómitos</a>, <a href="https://medlineplus.gov/spanish/abdominalpain.html">dolor abdominal</a> y <a href="https://medlineplus.gov/spanish/diarrhea.html">diarrea</a>. Los niveles más altos de hierro pueden causar <a href="https://medlineplus.gov/spanish/pepticulcer.html">úlceras</a>. Niveles extremadamente altos pueden provocar daño a los órganos, <a href="https://medlineplus.gov/spanish/coma.html">coma</a> y la muerte.</p>

<p>Una enfermedad llamada <a href="https://medlineplus.gov/spanish/hemochromatosis.html">hemocromatosis</a> puede provocar la acumulación de demasiado hierro en el cuerpo. La hemocromatosis se hereda (se transmite de padres a hijos). Por lo general, se trata extrayendo sangre (y hierro) del cuerpo de forma regular.</p>

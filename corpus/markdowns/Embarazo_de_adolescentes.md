# Embarazo de adolescentes

<p>La mayoría de las adolescentes no planea <a href="https://medlineplus.gov/spanish/pregnancy.html">embarazarse</a>, pero muchas lo hacen. Los embarazos en la adolescencia tienen riesgos adicionales de salud tanto para la madre como para el bebé. Con frecuencia, las adolescentes no reciben cuidados prenatales a tiempo, lo que puede provocar problemas posteriores. Tienen mayor riesgo de <a href="https://medlineplus.gov/spanish/highbloodpressureinpregnancy.html">hipertensión arterial</a> y sus complicaciones. Los riesgos para el bebé incluyen <a href="https://medlineplus.gov/spanish/prematurebabies.html">partos prematuros</a> y niños con poco peso al nacer. </p>
<p>Si eres adolescente y estás embarazada, puedes ayudarte y ayudar a tu bebé si:</p>
<ul>
<li>Obtienes cuidado durante el embarazo en forma regular</li>
<li>    Tomas las vitaminas prenatales para tu salud y para prevenir algunos defectos congénitos</li>
<li>Evitas el  <a href="https://medlineplus.gov/spanish/pregnancyandsubstanceuse.html">cigarrillo, el alcohol y las drogas</a></li>
<li>Usas un condón, en caso de que continúes teniendo relaciones sexuales, para prevenir infecciones de transmisión sexual que podrían dañar a tu bebé. Si eres o tu pareja es <a href="https://medlineplus.gov/spanish/latexallergy.html">alérgica al látex</a>, puedes usar condones de poliuretano </li>
</ul>

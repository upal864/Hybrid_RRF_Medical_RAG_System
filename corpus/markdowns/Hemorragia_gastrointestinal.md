# Hemorragia gastrointestinal

<p>Las vías intestinales o gastrointestinales incluyen el esófago, el estómago, el intestino delgado, el intestino grueso o colon, el recto y el ano. La hemorragia (sangrado) puede provenir de una o más de estas zonas. La cantidad de sangre puede ser tan pequeña que sólo se detecta con pruebas de laboratorio.</p>
<p>Los signos de hemorragia en el tracto digestivo dependen de dónde se encuentra y cuánto sangrando hay. </p>

<p>Los signos de sangrado en el tracto digestivo superior incluyen:</p><ul>
<li>Sangre de un color rojo brillante en el vómito</li>
<li>Vómito que parece café molido</li>
<li>Heces negras o alquitranadas</li>
<li>Sangre oscura mezclada con heces</li>
<li>Heces mezcladas o recubiertas de sangre de color rojo brillante </li>
</ul>

<p>Los signos de sangrado en el tracto digestivo inferior incluyen:</p><ul>
<li>Heces negras o alquitranadas</li>
<li>Sangre oscura mezclada con heces</li>
<li>Heces mezcladas o recubiertos de sangre de color rojo brillante</li>
</ul>

<p>La hemorragia gastrointestinal no es una enfermedad, sino el síntoma de una enfermedad. Existen muchas causas posibles de hemorragia gastrointestinal, entre ellas se encuentran <a href="https://medlineplus.gov/spanish/hemorrhoids.html">hemorroides</a>, <a href="https://medlineplus.gov/spanish/pepticulcer.html">úlceras pépticas</a>, desgarres o inflamación en el esófago, <a href="https://medlineplus.gov/spanish/diverticulosisanddiverticulitis.html">diverticulosis y diverticulitis</a>, <a href="https://medlineplus.gov/spanish/ulcerativecolitis.html">colitis ulcerativa o ulcerosa</a> y <a href="https://medlineplus.gov/spanish/crohnsdisease.html">enfermedad de Crohn</a>, <a href="https://medlineplus.gov/spanish/colonicpolyps.html">pólipos del colon</a> o cáncer de <a href="https://medlineplus.gov/spanish/colorectalcancer.html">colon</a>, <a href="https://medlineplus.gov/spanish/stomachcancer.html">estómago</a> o <a href="https://medlineplus.gov/spanish/esophagealcancer.html">esófago</a>. </p>

<p>La prueba que se utiliza con mayor frecuencia para conocer la causa de la hemorragia gastrointestinal se llama <a href="https://medlineplus.gov/spanish/endoscopy.html">endoscopía</a>. Se utiliza un instrumento flexible que se inserta a través de la boca o el recto para ver el interior de las vías intestinales (GI). Un tipo de endoscopía  que analiza el intestino grueso se llama <a href="https://medlineplus.gov/spanish/colonoscopy.html">colonoscopía</a>. </p>

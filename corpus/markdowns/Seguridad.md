# Seguridad

<p>Es imposible eliminar todos los peligros en la vida, pero se pueden disminuir. Si se toman las siguientes medidas, se pueden evitar los peligros mayores y prepararse para emergencias:</p><ul>
<li> Conserve los números telefónicos de emergencia cerca de sus teléfonos </li>
<li>	Prepare un botiquín de primeros auxilios para su casa </li>
<li>	Prepare un plan familiar de emergencia </li>
<li>	Instale y mantenga activados los detectores de <a href="https://medlineplus.gov/spanish/firesafety.html">humo</a> y de <a href="https://medlineplus.gov/spanish/carbonmonoxidepoisoning.html">monóxido de carbono</a> </li>
<li>	Mantenga las armas descargadas y bajo llave. Guarde las municiones separadamente bajo llave</li> 
<li>	Siga cuidadosamente las instrucciones cuando utilice herramientas o equipos</li>
</ul>
<p>Los <a href="https://medlineplus.gov/spanish/childsafety.html">niños</a> pequeños corren riesgos especiales. La supervisión es la mejor manera de mantenerlos seguros. Preparar la casa con protección especial para niños también puede ayudar.
</p>

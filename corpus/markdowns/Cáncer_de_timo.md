# Cáncer de timo

<p>El timo es un pequeño órgano ubicado en la parte superior del pecho, por debajo del esternón. Antes del nacimiento y en la infancia, el timo ayuda a producir un tipo de glóbulos blancos. Estas células ayudan a protegerlo contra las infecciones.</p>
<p>El cáncer del timo es poco frecuente. Tiene más probabilidades de tener uno si tiene otras enfermedades como <a href="https://medlineplus.gov/spanish/myastheniagravis.html">miastenia grave</a>, lupus o artritis reumatoide. Algunas veces no presenta síntomas. Otras veces, el cáncer de timo puede causar:</p>
<ul>
<li>	Tos persistente</li>
<li>Dolor en el pecho</li>
<li>	Dificultades para respirar </li>
</ul>
<p>Los médicos utilizan un examen físico, pruebas de imagen y una biopsia para diagnosticar el cáncer de timo. El tratamiento más común es la cirugía para extirpar el tumor. Otras opciones incluyen la radioterapia, quimioterapia y terapia hormonal.</p>
